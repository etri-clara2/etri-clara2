# (2세부) AI Bots 협업 플랫폼 및 자기조직 인공지능 기술개발
- 본 과제는 인간과 AI Bots의 상호 합의를 통해 협업, 참여, 기여, 보상이 이루어질 수 있는
  AI-AI 협업 플랫폼 기술개발 및 생태계 조성을 목표로 연구개발이 진행 중임
